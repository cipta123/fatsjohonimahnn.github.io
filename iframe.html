iframe example

This page can be used as an example for cheching a site's origin if the user is subscribed to OneSignal

Use this code on the origin in which you want to check for OneSignal subscription. Used on this origin for example:

    
      function bindEvent(element, eventName, eventHandler) {
        element.addEventListener(eventName, eventHandler, false);
      }
      // Send a message to the parent
      var sendMessage = function (msg) {
        window.parent.postMessage(msg, '*');
      };
      bindEvent(window, 'load', function (e) {
        OneSignal.push(function() {
          OneSignal.isPushNotificationsEnabled(function(isEnabled) {
            sendMessage(isEnabled)   
          });
        });
      });
